📝 Lesson 1: Formatting Text with CSS Typography

What is Typography?

Typography refers to the styling and arrangement of text on a web page using Cascading Style Sheets. It encompasses all the properties and techniques used to control how text appears to the user, ensuring legibility, readability, and visual appeal.

🔤 Font Properties

font-family

Specifies the typeface (example: Arial, "Times New Roman")

font-family: "Source Sans Pro", "Arial", "Sans-Serif";

The browser will try to use Source Sans Pro if available. If not found, it will try Arial. If that's not available either, it will use the browser's sans-serif font.

font-size

Sets the size of the text

font-size: 12px;
font-size: 12pt;
font-size: 50%;
font-size: medium;

Values: normal, xx-small, x-small, small, medium, large, x-large, xx-large, or any number followed by px, pt, or %

font-weight

Defines the weight, boldness or thickness of the text

font-weight: bold;
font-weight: 200;

100 - Thin

200 - Extra Light

300 - Light

400 - Normal

500 - Medium

600 - Semi Bold

700 - Bold

800 - Extra Bold

900 - Ultra Bold

font-style

Controls whether the text is normal, italic, or oblique

font-style: italic;

Values: italic, oblique, normal

font-variant

Defines which glyph to use for each letter (small-caps or other variations)

font-variant: normal;
font-variant: small-caps;

Glyph refers to the specific visual representation of a character or symbol within a font.

📏 Text Properties

line-height

Adjusts the vertical space between lines of text

line-height: 11px;

letter-spacing

Controls the horizontal space between individual characters

letter-spacing: 2px;

text-align

Specifies the horizontal alignment of text

text-align: justify;

Values: left, right, center, justify

text-indent

Indents the first line of a text block

text-indent: 25px;

color

Specifies the text color

color: violet;

word-spacing

Defines the spacing between words

word-spacing: normal;

word-break

Defines how words should break when reaching the end of a line

word-break: normal;

Values: normal, break-all

✨ Text Decoration & Transformation

text-decoration

Adds lines to text (underline, overline, line-through)

text-decoration: underline;

Values: none, underline, overline, line-through, blink

text-transform

Changes the capitalization of text

text-transform: lowercase;

Values: none, capitalize, uppercase, lowercase

text-shadow

Defines the shadow of the text content

text-shadow: none;
text-shadow: 2px 6px;
text-shadow: 2px 6px red;

The first value is horizontal offset, the second is vertical offset. You can define a color as the last value.

🔲 White Space Property

Defines how the element's white space is handled

normal: Sequences of spaces are combined into one. Line breaks are ignored. Text content is wrapped.

nowrap: Sequences of spaces are combined into one. Line breaks are ignored. Text remains on a single line.

pre: White space is exactly preserved. Sequences of spaces are preserved. Lines break only on new lines.

pre-wrap: White space is mostly preserved. Lines break on new lines and when reaching the end of the element.

pre-line: Only new lines are preserved. Sequences of spaces are combined. Lines break on new lines and at the end of the element.

📝 Lesson 1A: Advanced Typography

🔤 Font-Variant Deep Dive

normal

This is the default value. It specifies a regular, non-small-caps font face.

small-caps

This value converts all lowercase letters in the text to uppercase letters. However, these converted uppercase letters are rendered in a smaller size than the original uppercase letters present in the text.

This creates a visual effect where all letters appear capitalized, but the initial capital letters are slightly larger.

💫 Text-Shadow Advanced

This property allows you to add a shadow to any text. It works similarly to the box-shadow property but has only four values: h-shadow, v-shadow, blur, and color.

text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
text-shadow: 0 0 10px #0096FF, 0 0 20px #64C8FF;

📦 Word Wrap

The word-wrap property instructs the browser how to handle words that extend outside an element's boundary.

normal: Words will extend outside the element box if they are too long.

break-word: The browser will break the word to fit it into the element box.

word-wrap: normal;
word-wrap: break-word;

🎨 Lesson 2: Text Formatting & Borders

📝 Indenting Text

The first line of a paragraph can be indented using the text-indent property in CSS.

}

Measurement can be in px, mm, cm, in, pt, pc, ex, em, or %

🔤 Changing Text Case

Change the text case using the text-transform property in CSS.

p { text-transform: capitalize; }
span { text-transform: lowercase; }

Values: capitalize, uppercase, lowercase, none

📏 Changing Line Spacing

Change the line spacing using the line-height property in CSS.

p {

🎨 Adding Background Color to Text

Add a color behind text without changing the page's background using the background property in CSS.

span {
  background: yellow;

}

🔲 Adding a Border

You can add a border to your page using the border property in CSS. You can also add border styles such as solid, double, groove, ridge, inset, outset, dotted, or dashed.

CSS Border Style

dotted: Defines a dotted border

dashed: Defines a dashed border

solid: Defines a solid border

double: Defines a double border

groove: Defines a 3D grooved border (effect depends on border-color)

ridge: Defines a 3D ridged border (effect depends on border-color)

inset: Defines a 3D inset border (effect depends on border-color)

outset: Defines a 3D outset border (effect depends on border-color)

none: Defines no border

hidden: Defines a hidden border

div {
  border-style: solid;
  border-width: 2px;
  border-color: #0096FF;
}

📐 Border Width

The border-width property specifies the width or thickness of the four borders.

The width can be set as a specific size (px, pt, cm, em) or by using pre-defined values: thin, medium, or thick.

Multiple Values:

/* One value: all four borders */
border-width: 5px;

/* Two values: top/bottom, left/right */
border-width: 5px 10px;

/* Three values: top, left/right, bottom */
border-width: 5px 10px 15px;

/* Four values: top, right, bottom, left (clockwise) */
border-width: 5px 10px 15px 20px;

Individual Properties:

border-top-width: 5px;
border-right-width: 10px;
border-bottom-width: 15px;
border-left-width: 20px;

🎨 Border Color

The border-color property is used to set the color of the four borders.

The color can be set by:

  • name - specify a color name like "violet"
  • hex - specify a hex value like "#ff0000"
  • RGB - specify an RGB value like "rgb(255,0,0)"
  • HSL - specify an HSL value like "hsl(0,100%,50%)"
  • transparent
p.one {
  border-style: solid;
  border-color: red green blue yellow;
  /* red-top, green-right, blue-bottom, yellow-left */
}

📋 Creating Lists

In CSS, you can customize lists - choose your own image as bullets instead of pre-defined squares, circles, and discs, and control the indentation of items.

CSS List Properties:

list-style-type: Indicates the type of bullet

list-style-image: Indicates the image to be used as bullet

list-style-position: Indicates the position of the marker

list-style: Shorthand notation for all list properties

List Style Types:

ul { list-style-type: disc; } /* round bullets */
ul { list-style-type: circle; } /* circular bullets */
ul { list-style-type: square; } /* square bullets */
ol { list-style-type: decimal; } /* 1, 2, 3, 4... */
ol { list-style-type: decimal-leading-zero; } /* 01, 02, 03... */
ol { list-style-type: lower-roman; } /* i, ii, iii, iv... */
ol { list-style-type: upper-roman; } /* I, II, III, IV... */
ol { list-style-type: lower-alpha; } /* a, b, c... */
ol { list-style-type: upper-alpha; } /* A, B, C... */

Custom Image Bullets:

ul {
  list-style-image: url('bullet.png');
}

✨ Pseudo-elements

Pseudo-elements allow you to format a specific part or several parts of HTML elements. You can apply design using CSS properties on the first letters and first lines of text.

CSS Pseudo-Elements:

::first-letter - Selects the first letter of every element

::first-line - Selects the first line of every element

::before - Inserts content before every element

::after - Inserts content after every element

p::first-letter {
  font-size: 200%;
  color: #0096FF;
}

p::first-line {
  font-weight: bold;
}

h1::before {
  content: "⚡ ";
}

h1::after {
  content: " ⚡";
}

📦 Lesson 3: Box Model & Layout

📦 Understanding the Box Model

To understand layout using CSS, you have to use the box model where each element on a page exists in its own box wherein each box aligns with other boxes on the page.

The box model consists of:

  • Margins - Space outside the border
  • Borders - The edge of the box
  • Padding - Space between content and border
  • Content - The actual content (text, images, etc.)

CONTENT

Padding

Border

Margin

📍 Element Positioning

Position property allows placing an object or element precisely on a page. There are four position schemes:

static

The default or normal position of an object if one would not specify a position.

relative

The position of the box relative to where the default or normal position of an object.

absolute

The position is defined by the values from the specified reference point.

fixed

Same as absolute, but the object is "fixed" and will not move with scroll action.

.static { position: static; }
.relative { position: relative; top: 100px; left: 100px; }
.absolute { position: absolute; top: 100px; left: 100px; }
.fixed { position: fixed; top: 100px; left: 300px; }

📏 Offset Properties

Offset property allows you to set up the distance of the specified element from another element. This will allow you to place contents of your page in a precise position within the browser window.

Offset property works together with the position property. Before you can use an offset property, you have to set up the position property first.

top: Sets how far the box's top content is offset from another element

right: Sets how far the box's right content is offset from another element

bottom: Sets how far the box's bottom content is offset from another element

left: Sets how far the box's left content is offset from another element

.positioned {
  position: absolute;
  top: 50px;
  left: 100px;
}

🌊 Float Property

Float property allows you to control where a specified block element will be located or placed on the page and where the other elements will be positioned in relation to the said block.

When using the float property, it is important to set the width of the block for better control of the float.

left: The specified block element "floats" to the left and all other elements near it will float away from the block.

right: The specified block element "floats" to the right and all other elements near it will float away from the block.

none: The specified block element will take the space it requires and nothing flows to the right or left of it.

img {
  float: left;
  margin-right: 15px;
}

.sidebar {
  float: right;
  width: 300px;
}

🖼️ Wrapping Text around an Element

We can wrap text around an element by using the float property in CSS.

img {
  float: left;
  margin: 0 15px 10px 0;
}

🎨 Background Image

We can specify a background image on any element by setting the location as specified in the url().

body {
  background-image: url('background.jpg');
}

🔁 Background Repeat

Background image can be repeated by using the background-repeat property in CSS. You can control how an image background should repeat in order to cover the whole area of a web page or the desired area.

repeat: Repeats background (default)

repeat-x: Repeats background horizontally

repeat-y: Repeats background vertically

no-repeat: Do not repeat the background

inherit: Follow the parent's repeat setting

body {
  background-image: url('pattern.png');
  background-repeat: repeat-x;
}

📍 Background Position

Background can be located where the image will be positioned on the web page by specifying a certain value or keyword location.

/* Using pixel values */
background-position: 100px 50px;

/* Using percentages */
background-position: 20% 10%;

/* Using keywords */
background-position: top left;
background-position: center;
background-position: bottom right;

📌 Background Attachment

This property allows you to control how the background scrolls with the web page. There are three types:

scroll: Background scrolls with document (default setting)

inherit: Follow parent's attachment

fixed: Background is steady and does not follow scroll action

body {
  background-image: url('bg.jpg');
  background-attachment: fixed;
}

📐 Setting Margins

Margins can be set in your webpage by using the margin property in CSS. You can use the values for the top, bottom, left and right margins around an element.

margin-top: Sets the top margin

margin-bottom: Sets the bottom margin

margin-left: Sets the left margin

margin-right: Sets the right margin

margin: Shorthand for all margins

/* Individual margins */
margin-top: 10px;
margin-bottom: 10px;
margin-left: 5px;
margin-right: 5px;

/* Shorthand - all sides */
margin: 15px;

/* Shorthand - top/bottom, left/right */
margin: 10px 20px;

/* Shorthand - top, left/right, bottom */
margin: 10px 20px 15px;

/* Shorthand - top, right, bottom, left */
margin: 10px 20px 15px 5px;

If you place only one value on margin, that value will be applied to all sides. If you do not specify a margin, the default value is zero. A negative margin value can also be used to get special effects.

📋 Padding Property

Padding indicates the distance between the contents of a cell and the border around it. Padding in HTML works only on tables while the padding provided by CSS can be used on other elements as well as paragraphs, headings, images, etc.

padding-top: Distance between contents and top border

padding-bottom: Distance between contents and bottom border

padding-left: Distance between contents and left border

padding-right: Distance between contents and right border

padding: Distance between contents and border (clockwise: top, right, bottom, left)

/* Individual padding */
padding-top: 12%;
padding-bottom: 12px;
padding-left: 12%;
padding-right: 12px;

/* Shorthand */
padding: 12px; /* all sides */
padding: 10px 20px; /* top/bottom, left/right */
padding: 10px 20px 15px; /* top, left/right, bottom */
padding: 10px 20px 15px 5px; /* top, right, bottom, left */

If you specify only one value of padding, that value will be applied equally along the sides of the content. If you do not specify padding, the default value is zero.
