💻 ICT / TLE Lessons

Lesson 1: What is CSS?

CSS (Cascading Style Sheets) is used to style and design webpages by controlling how HTML elements appear. It separates structure (HTML) from design (CSS), making websites easier to maintain, update, and beautify.

Benefits:

  • Saves time – styles can be reused across pages
  • Improves consistency and maintainability
  • Enables responsive and accessible design
  • Enhances visual presentation (colors, layout, fonts)

Webpage Layers:
  1. Content Layer – HTML structure (text, images, media)
  2. Presentation Layer – CSS styling and layout
  3. Behavior Layer – JavaScript interactivity

Example: p { color: red; } .highlight { background-color: yellow; }

Lesson 2: CSS Syntax and Structure

CSS consists of rules that tell the browser how to style HTML elements. Each rule is made up of a selector and a declaration block.

CSS Rule Structure:

  • Selector – specifies the HTML element(s) to style (e.g., p, .class, #id)
  • Declaration Block – wrapped in curly braces { } containing one or more declarations
  • Declaration – property and value pairs that define the style, separated by colons and ended with semicolons, e.g. color: red;

Example of a CSS Rule: p {
  color: blue;
  font-size: 18px;
}
Key CSS Components:
  • { } – curly braces group declarations for a selector
  • : – separates property and value
  • ; – ends each declaration
  • Selectors can target elements by type, class, or id
  • Declarations define what style changes to make

Summary: CSS syntax is straightforward once you remember the pattern: selector { property: value; } — used repeatedly to style webpages.

Lesson 3: Display Property

The display property controls how an element appears on the page.

Common values include:

  • block: starts on a new line
  • inline: stays on the same line
  • none: hides the element

Examples: Block display: div {
display: block;
width: 100%;
background-color: lightgray;
}
Inline display: a {
display: inline;
color: blue;
}
None display: .hidden {
display: none;
}
Summary: The display property is essential in controlling the layout and behavior of HTML elements. Block-level elements create new lines, inline elements do not, and display set to none removes the element from the page layout entirely.

Lesson 4: CSS Rule

A CSS rule has two main parts: a selector, and one or more lists of style declarations.

Basic Format of a CSS Style Rule: For Inline Style Rule: <tag style="style declaration">
For Document-Level and External Style Sheets: selector {style declarations}

Example: p {
color: blue;
font-size: 18px;
}
Summary: CSS rules define how HTML elements should be styled. The basic structure consists of a selector and a declaration block. Each declaration defines a specific style for the selected element(s).

Lesson 5: CSS Selectors

CSS selectors are used to "find" or "select" the HTML elements that you want to style.

CSS selectors are divided into six types:

  1. Class selector
  2. Element selector or Tag selector
  3. ID selector
  4. Universal selector
  5. Group selector
  6. Attribute selector

Element Selector

The element selector selects HTML elements (like p, div, h1, etc.) and applies CSS to them.

ID Selector

The ID selector is used to style a unique element on the page, typically in cases such as page titles or navigation sections.

Universal Selector

The universal selector selects every single HTML element on the page. It is represented by the * symbol.

Group Selector

The group selector allows you to select multiple elements and apply the same style to all of them.

Attribute Selector

The attribute selector selects elements based on specific attribute values. The syntax is Element[attribute].

Summary:

CSS selectors are powerful tools to select and style specific HTML elements. Whether you're applying styles to a specific tag, using unique IDs, selecting elements by their attributes, or grouping multiple elements, CSS selectors provide flexibility and control in styling your webpage.

Lesson 6: Setting Dimensions

CSS can be used to manage dimensions such as visibility, width, height, and more for HTML elements.

CSS properties for dimensions include:

  1. Visibility: Indicates if an element is visible or hidden.
  2. Width: Specifies the width of an element.
  3. Height: Specifies the height of an element.
  4. Line-height: Defines the spacing between lines of text.
  5. Max-height: Sets the maximum height an element can have.
  6. Min-height: Defines the minimum height an element must have.
  7. Max-width: Specifies the maximum width an element can have.
  8. Min-width: Specifies the minimum width an element must have.

Visibility

The visibility property determines if an element is visible or hidden, but it still takes up space in the layout.

  • visibility: visible – Default value, makes the element visible.
  • visibility: hidden – Hides the element but it still occupies space.
  • visibility: collapse – Only for table rows, columns, and flex items; effectively removes the element.

Width & Height

Width and Height properties define the size of an element. You can set fixed dimensions or let the element resize automatically.

  • width: 200px – Sets the width of the element to 200px.
  • height: auto – Sets the height to auto, based on content.

Max & Min Dimensions

Max-width, Min-width, Max-height, and Min-height properties are used to define the maximum or minimum size of an element.

  • max-width: 12px – Limits the width to 12px.
  • min-width: 12px – Ensures the width is at least 12px.
  • max-height: 12px – Limits the height to 12px.
  • min-height: 12px – Ensures the height is at least 12px.

Summary:

CSS provides several ways to control the dimensions of elements on a webpage. By using properties like width, height, visibility, and their related properties, you can design flexible and responsive layouts.

📚 Araling Panlipunan Lessons: Yunit II - Maykroekonomiks

Deskripsyon: Ang demand ay ang dami ng produkto o serbisyo na kaya at handang bilhin ng mamimili sa isang tiyak na presyo.

Batas ng Demand (Ceteris Paribus): Kapag ang presyo ay tumaas, bababa ang demand. Kapag ang presyo ay bumaba, tataas ang demand.

Substitution Effect: Kapag ang presyo ng isang produkto ay tumaas, hahanap ang mamimili ng mas murang alternatibo.

Income Effect: Kapag tumaas ang presyo, parang lumiit ang kita ng mamimili kaya bumababa ang demand.

Mathematical Formula: Qd = a - bP

Demand Schedule: Talaan na nagpapakita ng presyo at quantity demanded.

Demand Function: Qd = f(P)

Demand Curve: Graph na pababa mula kaliwa-pakanan.

Mga Salik na Nakaaapekto: Kita, Panlasa, Populasyon, Ekspektasyon, Presyo ng kaugnay na produkto.

Deskripsyon: Dami ng produkto/serbisyo na kayang ipagbili ng producer sa tiyak na presyo.

Batas ng Supply: Kapag mataas ang presyo, tumataas ang supply; kapag mababa, bumababa ang supply.

Ceteris Paribus: Ipinapalagay na ang ibang salik ay hindi nagbabago.

Mathematical Formula: Qs = c + dP

Supply Schedule: Talaan ng presyo at quantity supplied.

Supply Function: Qs = f(P)

Supply Curve: Graph na pataas mula kaliwa-pakanan.

Mga Salik na Nakaaapekto: Teknolohiya, Gastos, Ekspektasyon, Bilang ng nagbebenta.

Deskripsyon: Sukatan kung gaano ka-sensitibo ang demand sa pagbabago ng presyo.

Elasticity ng Demand (ED): ED = % pagbabago sa Qd / % pagbabago sa presyo

Uri ng Elastisidad:

  • Elastic: ED > 1
  • Di-Elastic: ED < 1
  • Unitary: ED = 1
  • Ganap na Elastik: ED = ∞
  • Ganap na Di-Elastik: ED = 0

Deskripsyon: Punto kung saan nagtatagpo ang demand at supply sa isang presyo at quantity.

Equilibrium Price: Presyong nagbabalanse sa dami ng demand at suplay.

Equilibrium Quantity: Dami ng produkto na nais bilhin at ibenta sa equilibrium price.

Shortage: Kapag mas mataas ang demand kaysa supply.

Surplus: Kapag mas mataas ang supply kaysa demand.

Deskripsyon: Uri ng pamilihan ayon sa bilang ng producer at katangian ng produkto.

  • Ganap na Kompetisyon: Maraming nagbebenta, pare-pareho ang produkto.
  • Monopolyo: Iisa ang producer.
  • Oligopolyo: Iilang producer, nagkokontrol ng presyo.
  • Monopolistikong Kompetisyon: Marami ang nagbebenta, ngunit may pagkakaiba sa produkto.

Deskripsyon: Layunin ng pamahalaan na mapanatili ang balanse sa ekonomiya sa pamamagitan ng regulasyon.

Price Control: Pagkontrol ng pamahalaan sa presyo ng mga produkto upang hindi ito magmahal nang sobra.

RA 7581: Price Act na naglalayong protektahan ang mamimili laban sa pananamantala sa presyo.

Price Ceiling: Pinakamataas na presyong maaaring singilin.

Price Support: Suportang presyo para sa mga prodyuser.

Floor Price: Pinakamababang presyong pinapayagan sa isang produkto.
